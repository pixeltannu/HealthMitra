# HealthMitra 🩺

A web-based health assistant built for **Bharat Academix CodeQuest Hackathon 2026**.

## About

HealthMitra helps users in rural and semi-urban areas of India by:

1. **Symptom Checker** – Users select their symptoms (fever, cough, headache, body ache, etc.) and the system suggests possible conditions and whether they should consult a doctor or rest at home, using rule-based logic.
2. **Nearby Health Center Locator** – Users select their area to view nearby government hospitals, PHCs, CHCs, and Jan Aushadhi stores along with address and contact details.
